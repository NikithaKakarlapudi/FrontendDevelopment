Badge

1.Badges can be created very easily by adding .badge class along with any contextual class (like .badge-primary) to the element base class.
2.The followings are the badge contextual classes: .badge-primary, .badge-success, .badge-secondary, .badge-warning, .badge-danger, .badge-info, .badge-light, .badge-dark.

To create a pill badge

1.Add .badge followed by .badge-pill to the element base class.
2.Add one of the badge contextual classes to the element base class.
3. The followings are the predefined badge contextual classes: .badge-primary, .badge-success, .badge-secondary, .badge-warning, .badge-danger, .badge-info, .badge-light, .badge-dark.
